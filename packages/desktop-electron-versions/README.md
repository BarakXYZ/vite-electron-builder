# @app/desktop-electron-versions

Helper utilities for reading the Electron, Chromium, and Node.js versions bundled with the desktop app.

## Install

```bash
pnpm add -D @app/desktop-electron-versions
```

## Usage

```js
import { getChromeMajorVersion, getChromeVersion, getElectronVersions, getNodeMajorVersion, getNodeVersion } from "@app/desktop-electron-versions";
```
